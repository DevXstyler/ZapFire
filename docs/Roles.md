# Possible Roles for Enemys
### Basic Roles
- pawn (basic unit)
- archer (ranged attacker)
- thief (fast and agile)
### Advanced Roles
- executioner (high damage dealer)
- tank (defensive role)
- Barbarian (strong, fast, durable)
- healer (support role)
- monk (can protect allies)
- mage (magic user, attacks like fireball, lightning, etc.)
### Elite Roles
- knight (strong melee fighter with armor and abilities)
- witch (can cast curses)
- Oni (Demonic warrior with high strength and magic)
### Boss Mobs
- necromancer (can raise the dead) 

## Summonable Roles/creatures
- golem (extremely durable unit)
- dwarf (fast and many lives)
- goblins (swarm attackers)

### Undead Roles
- skeleton (basic undead)
- zombie (slow but durable)
- ghost (can phase through walls)

# Possible Roles for Players
- warrior (melee fighter)
- rogue (can sneak attack)
